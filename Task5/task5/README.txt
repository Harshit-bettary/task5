This project is a simple ReactJS application that fetches product data from the Fake Store API
and allows users to add products to the cart through a modal interface.

Tech Stack Used:
ReactJS
HTML/CSS
JavaScript for functionality

Fetch products from the Fake Store API.
Display products with image, title, and price.
Add products to the cart using the Add to Cart button.
Alert message if the product is already added to the cart.
Cart items count displayed in the Navbar.
Cart modal that displays all added products.
Option to Remove from Cart.
Responsive design for both desktop and mobile devices.
